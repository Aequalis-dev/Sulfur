# Luau Script Bot

A Discord bot for your friend group's Roblox server. Type `/script <describe what you want>` and it replies with clean, efficient Luau code, generated by Claude.

---

## 1. Create the Discord bot

1. Go to https://discord.com/developers/applications
2. Click **New Application**, name it (e.g. "Luau Bot"), create it.
3. In the left sidebar, click **Bot**.
   - Click **Reset Token** (or **Add Bot** if prompted), then copy the token. This is your `DISCORD_TOKEN`. Keep it secret — anyone with it controls your bot.
4. Still on the **Bot** page, turn off "Public Bot" if you don't want strangers adding it.
5. Go to **OAuth2 > General** and copy the **Client ID**. This is your `CLIENT_ID`.
6. Go to **OAuth2 > URL Generator**:
   - Check **bot** and **applications.commands** under Scopes.
   - Under Bot Permissions, check **Send Messages**, **Use Slash Commands**, **Embed Links**, **Attach Files**.
   - Copy the generated URL, open it in your browser, and invite the bot to your server.

## 2. Get a free OpenRouter API key

1. Go to https://openrouter.ai and sign up (no credit card needed).
2. Create an API key under **Keys** in your dashboard.
3. This is your `OPENROUTER_API_KEY`.
4. The bot defaults to `qwen/qwen3-coder:free` — a strong free coding model. Free-tier models are rate-limited (roughly 20 requests/minute, 200/day), which is plenty for a friend group server. If you want a different free model, change `OPENROUTER_MODEL` in your `.env` (other options: `deepseek/deepseek-r1-distill:free`, `meta-llama/llama-3.3-70b:free`).

## 3. Configure the bot

1. Copy `.env.example` to `.env`:
   ```
   cp .env.example .env
   ```
2. Fill in `DISCORD_TOKEN`, `CLIENT_ID`, and `OPENROUTER_API_KEY` in `.env`. `OPENROUTER_MODEL` is already set to a good default.

## 4. Run it locally (to test)

```bash
npm install
npm start
```

If it connects successfully, you'll see `Logged in as <BotName>` in the console. Then in your Discord server, try:

```
/script make a script that lets players double jump
```

## 5. Host it permanently

Running `npm start` on your own computer only works while your computer is on. To keep it running 24/7, deploy it to a server. See the earlier recommendation for a free option:

- **Oracle Cloud Free Tier** (a real always-free VM) — set up a VM, install Node.js 18+, upload this folder, run `npm install`, then keep it alive with:
  ```bash
  npm install -g pm2
  pm2 start index.js --name luau-bot
  pm2 save
  pm2 startup
  ```
  This restarts the bot automatically if it crashes or the server reboots.

- Alternatively: Railway, Fly.io, or a cheap VPS (DigitalOcean/Hetzner) work the same way — install Node, copy the files, run with pm2 or a systemd service.

## Notes

- Generated scripts over ~1900 characters are sent as a `.lua` file attachment instead of inline, so long scripts aren't cut off.
- The bot uses OpenRouter's free tier — no cost, but rate-limited (~20 requests/minute, 200/day as of writing). Free model availability and limits can change without notice; check https://openrouter.ai if requests start failing.
- You can tweak the system prompt in `index.js` (the `SYSTEM_PROMPT` constant) to change the coding style or add house rules (e.g. always add type annotations, always use a specific module pattern).
- If your friend group outgrows the free tier, add a few dollars of credit to your OpenRouter account and switch `OPENROUTER_MODEL` to a paid model — no other code changes needed.
